# Assembly [27 %]
Threads on the parts are designed to tighten into each other.<br>
Every thread is right-handed M3*0.5, which means you need to rotate it CLOCKWISE to tighten the parts.
Follow the instruction below to complete your hotend.
## Level 0
The order of the throat parts is THROAT2 - THROAT0 - THROAT1.<br>
Screw THROAT2 and THROAT1 to THROAT0 by rotating them clockwise.
